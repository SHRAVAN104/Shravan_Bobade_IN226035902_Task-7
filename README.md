# Shravan_Bobade_IN226035902_Task-7

This project contains solutions to 6 object-oriented programming problems implemented using Python classes.

- Constructors (__init__)
- Object instantiation
- Method definition
- Real-world system modeling

## Problems Covered

2. Employee ID Card System  
3. Mobile Contact Record  
4. Product Price Tag Generator  
5. Movie Rating Display System

Each problem demonstrates basic OOPS principles for real-world applications.
